Probability: Union and Intersection of Events

Grade 10 Mathematics - 3rd Quarter

I. Understanding Sets in Probability

Probability often deals with the relationship between different events within a sample space. We visualize these relationships using Venn Diagrams.

1. Union of Events (A ∪ B)

The Union represents the event that occurs if A occurs, OR B occurs, OR both occur. In a set, it combines all unique elements from both groups.

2. Intersection of Events (A ∩ B)

The Intersection represents the event that occurs only if BOTH A and B occur at the same time. It is the overlapping region in a Venn diagram.

II. The Addition Rule Formula

To calculate the probability of the union of two events, we use the General Addition Rule:

P(A ∪ B) = P(A) + P(B) - P(A ∩ B)

Why the subtraction? If we simply added P(A) and P(B), the outcomes in the intersection (the overlap) would be counted twice. Subtracting the intersection once ensures an accurate total.

III. Mutually Exclusive Events

Events are Mutually Exclusive if they cannot happen at the same time. For example, when rolling a single die, you cannot roll a '2' and a '5' simultaneously.

For these events: P(A ∩ B) = 0, so the formula simplifies to:
P(A ∪ B) = P(A) + P(B).
